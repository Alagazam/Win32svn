# **Subversion for Windows**

**Command line tools, Language bindings, and Apache httpd modules**

### Win32 build of Subversion

These binaries are built using Visual C++ 6.0 Should work on all flavours of Windows from Win2000 to Win8 and 2008 Server including server variants (not all tested).
(1.7.x does not work on NT4 due to APR using new functions).

Modules for Apache 2.2.x and 2.4.x (1.7.6 and up) is included.

Language bindings are NOT tested.

Source code can be found at the Apache Subversion site at [https://subversion.apache.org/](http://subversion.apache.org/)

Code in this project is just a "Build script" and patches for VC6

## Features

* Subversion command line client
* Subversion server
* Apache 2.2 and 2.4 modules
* Language bindings (not tested)
